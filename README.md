# CampusConnect 🚀

### Smart Campus Ambassador Management Platform

🌐 **Live Demo:** https://campus-connect-three-lovat.vercel.app/

🎥 **Demo Video:** https://drive.google.com/file/d/1wuiAIj0QqKmbN3cPUTTR1C9SYGTaUagJ/view?usp=drive_link

CampusConnect is a modern web platform built for the AICore Connect Hackathon to help organizations efficiently manage Campus Ambassador programs across colleges.

It replaces spreadsheets, forms, and chat groups with a centralized platform for onboarding ambassadors, assigning tasks, tracking submissions, rewarding top performers, and measuring engagement through analytics.

## ✨ Features

* Admin Dashboard
* Task Management
* Ambassador Portal
* Leaderboards & Rewards
* Analytics Dashboard
* AI Insights
* Responsive Design
* Dark / Light Mode

## 🛠 Tech Stack

* React
* Vite
* React Router
* Chart.js
* Stitch MCP
* Antigravity
* Vercel

## 🎯 Problem Solved

CampusConnect transforms fragmented ambassador programs into scalable growth engines through automation, gamification, and analytics.
